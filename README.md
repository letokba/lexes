# lexes

a simple Four Operations Expression Analyzer.

## Test Samples
```text
    "1 + 2",                            // 3
    "2 + 3 * 2",                        // 8
    "3 - 2 / 2 * 4 + 4",                // 3
    "(1 + 2)",                          // 3
    "(1 + 2) * 3",                      // 9
    "(2 - 1) + (3 + 1)",                // 5
    "(2.5 - 0.5) * 2",                  // 4
    "((3.0 + 2) * 2) + 0 + (2 * 3)"     // 16
    "3 + (2 + 3) * 3"                   // 18

```

## Test Result

```text

    1 + 2 = 3.0
    2 + 3 * 2 = 8.0
    3 - 2 / 2 * 4 + 4 = 3.0
    (1 + 2) = 3.0
    (1 + 2) * 3 = 9.0
    (2 - 1) + (3 + 1) = 5.0
    (2.5 - 0.5) * 2 = 4.0
    ((3.0 + 2) * 2) + 0 + (2 * 3) = 16.0
    3 + (2 + 3) * 3 = 18.0

```
